# **極簡風番茄工作法時鐘 (Minimalist Pomodoro Timer)**

這是一個基於極簡主義美學設計的網頁番茄鐘應用程式。專為提升專注力、減少視覺干擾而設計，具備乾淨的介面與流暢的互動邏輯。

## **核心功能**

* 精準倒數計時：預設專注時間為 25 分鐘，休息時間為 5 分鐘。  
* 嚴謹控制邏輯：具備「開始」、「暫停」與「重置」功能，計時進行中會自動鎖定開始按鈕，防範重複觸發。  
* 動態網頁標題：倒數時網頁標題（Tab）會動態顯示剩餘時間；時間結束時會更新標題提醒，並播放輕柔提示音。  
* 數據本地暫存：利用瀏覽器 LocalStorage 技術，低調記錄並顯示使用者今日完成的番茄鐘總數。  
* 極簡視覺切換：介面全面使用繁體中文，支援深色模式 (Dark Mode) 與淺色模式 (Light Mode) 切換（自動跟隨系統設定）。

## **技術堆疊**

* HTML5：語意化網頁結構設計。  
* CSS3：極簡風排版、大量留白設計、響應式佈局與主題顏色變數。  
* Vanilla JavaScript (原生 JS)：負責計時核心邏輯、LocalStorage 資料儲存與 DOM 事件監聽。

## **如何在本地端執行**

為了極致的便利性與可攜性，本專案將所有的 HTML 結構、CSS 樣式與 JavaScript 邏輯整合在單一個 .html 檔案中。

1. 將本專案的 index.html 檔案下載到你的電腦中。  
2. 雙擊該 index.html 檔案，即可直接在任何現代瀏覽器中開啟並使用番茄鐘。
